ご利用の環境では、COCOA動作状況チェッカーを利用できません。
下記の項目を確認し設定してください。
Android 6.0以降のバージョンが必要です
ご利用の環境を、Android 6.0以降にアップグレードしてください。
ご利用の端末でのOSアップグレードの可否は、端末のメーカーまでお問い合わせ下さい。
最新版のChromeが必要です
  1. Playストアを開きます。
  2. マイアプリ＆ゲームを開きます。
  3. 「全て更新」をタップしてアプリを更新します。
Chromeの設定で"テスト中機能"を有効にする必要があります
Chromeの"experimental-web-platform-features"(テスト中機能)を"Enabled"(有効)にしてください。
  1. アドレス を開きます。
    ここをクリックすると、をクリップボードに格納して新しいウインドウを開くので、アドレス欄に「貼り付け」を行って下さい。
  2. 画面に表示される「Experimental Web Platform features」が、「Default」あるいは「Disabled」になっていますので、これを「Enabled」に変えます。
  3. 画面下部に現れる「Relaunch」というボタンをタップします。
Chromeの設定でBluetoothのスキャンを許可する必要があります
  1. アドレスバーにある鍵マークをタップして下さい。
  2. 「Bluetoothのスキャン - ブロック中」と表示されている場合には、
    1. 「サイトの設定」をタップしてください。
    2. 「データを削除してリセット」をタップしてください。
  3. 「Bluetoothのスキャン - ブロック中」と表示されていない場合には、
    1. 戻るボタンをタップしてください。
    2. 「…」をタップし「設定」をタップしてください。
    3. 「サイトの設定」をタップしてください。
    4. 「Bluetoothのスキャン」をタップしてください。
    5. スイッチをタップし、「サイトから近くにあるBluetoothデバイスの検出を求められた時に確認する（推奨）」と表示されるようにしてください。
BluetoothをOnにして、Chromeに位置情報の利用を許可する必要があります
  1. 端末のBluetoothのOn/Offを確認してください。
  2. BluetoothがOffになっていたらOnにしてください。
  3. BluetoothがOnになっていたら、Chromeの位置情報利用が許可されていませんので、
    1. Androidの「設定」を開きます。
    2. 「アプリ」を開きます。
    3. 「Chrome」を探して開きます。
    4. 「権限」をタップし、「位置情報」を許可します。
Bluetoothのスキャンを再度行ってください
